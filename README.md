# Help-Nearby

## Description
Help-Nearby is a web-based platform that connects users with nearby medical or support services. It aims to provide quick access to help in emergency or daily needs.

## Features
- Find nearby help/services
- User-friendly interface
- Fast access to important resources

## Tech Stack
- Python (Django)
- HTML, CSS
- SQLite (default Django DB)

## Project Structure
- templates/ → frontend HTML files
- vaidya/ → main project settings
- vaidya_app/ → application logic
- manage.py → project entry point

## How to Run
1. Install Python
2. Install dependencies:
   pip install django

3. Run server:
   python manage.py runserver

4. Open in browser:
   http://127.0.0.1:8000/
